Yelp Camp

Yelp Camp was a web application I developed and deployed. Developed using HTML, CSS, JavaScript, Bootstrap 4, Git, MongoDB, Express and NodeJS. This project was initially developed during the Udemy course, and after finishing that, I started refactoring and adding new functionalities, such as using google maps API and searching.

This is a work in progress, as I still have plans to add some other functionalities, such as user profiles and user ratings.

You can use "test" as username and password for testing the application.

Color Game

Color Game was a very simple application that I created to test and practice my skills on basic HTML, CSS and JavaScript. The game generates 6 random colors, shows you one of those chosen randomly on RGB format, and the player tries to guess which of those colors is the right one.

This application helped understand better the interaction of basic HTML, CSS and JavaScript. Using JavaScript to work with HTML elements and styling them and getting input from the user and changing the page.
